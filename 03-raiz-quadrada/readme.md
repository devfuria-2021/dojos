## Dojo

Utilizando funções, faça um programa que calcule as raízes da equação do 2 grau conforme a fórmula 
de Bhaskara.

   ax(2) + bx + c = 0

   delta = (Δ = b2 - 4.a.c)
   x1    = ( (-b + √Δ)/2a)
   x2    = ( (-b - √Δ)/2a)

Exemplo:

a =   1;
b =   0;
c = -16;

Delta = 64
x1    =  4
x2    = -4
